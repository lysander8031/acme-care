How to configure transactions in Cucumber
	- add @txn to .feature files 
	- add cucumber-spring dependency to pom.xml
	- add the following assert inside of the given step definition to test: 
		assertTrue("No transaction is active", TransactionSynchronizationManager.isActualTransactionActive());
		
How to configure transactions in Serenity with Cucumber
	 - add @txn to .feature files
	 - create SpringTransactionHooks.java
	 - annotate SpringTransactionHooks with @ContextConfig so dependencies could be injected
	 - add glue to @CucumberOptions - glue = {"com.acme.care.features", "com.acme.care.features.hooks"}
